# Electronic Health Records (EHR) Management System

![EHR Logo](ehr_image.png)  <!-- Include your generated image here -->

## Project Overview

Welcome to the Electronic Health Records (EHR) Management System! This project aims to provide a simple yet effective way to manage electronic health records for healthcare professionals and organizations.

## Features

- Add new patient records with patient ID, name, date of birth, and diagnosis.
- Update existing patient records with ease.
- Retrieve patient records based on their unique patient ID.
- Delete patient records securely.
- Generate random patient data for testing purposes.

## Getting Started

To use this EHR Management System, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/trapzzy/ehr-management.git
   cd ehr-management
   ```

2. Run the script:

   ```bash
   python ehr_management.py
   ```

3. Follow the on-screen instructions to interact with the system.
